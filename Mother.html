Mother is an interactive musical tool
A modular musical toy designed to encourage young kids to build and
modify their toys and environment.
Developed for the course T-experience, with two project partners.
We were instructed to develop a prototype of an interactive musical
instrument.

Our solution was a modular musical instrument, which can play
different types of sounds.

The game consists of a few soft modules: one 'mother' module, which
has several ports, and a few pillow shaped dongle modules, each with two
ports. Dongles can be connected to each other or to the mother module.

Pressing a dongle creates a sound; the relative position of the dongle to
the mother module defines the pitch, and the port connecting to the mother
module defines the instrument or sound.
